1. What this website stores

The Project Caligo website does not currently place cookies on your device or use local storage, session storage, advertising identifiers, or browser fingerprinting.

The contact form, Vercel Web Analytics, and Vercel Speed Insights work without placing a cookie or persistent measurement identifier in your browser.

2. How Vercel measurement works

Web Analytics records anonymous page-view data for aggregate statistics. A data point can include the page path, timestamp, referrer, broad geolocation, device type, operating system, and browser. Vercel states that it does not use third-party cookies or retain information that allows Project Caligo to identify or reconstruct an individual visitor's activity across websites. A temporary request-derived hash is discarded after 24 hours.

Speed Insights uses native browser performance APIs to report anonymous Core Web Vitals and related context. A data point can include the page path, network-speed category, browser, device type, operating system, country, the measured Web Vital, the page element associated with it, SDK information, and the server-received time. Vercel states that this data is not associated with an individual visitor or IP address and cannot reconstruct a browsing session across pages.

Our configuration removes query strings and URL fragments before measurement events are sent. More detail is available in Vercel's privacy documentation for Web Analytics and Speed Insights.

3. Why there is no consent banner

The current measurement configuration does not store or read cookies, local storage, or another persistent identifier on your device. A cookie accept-or-reject banner would therefore describe a browser-storage choice that this website does not use.

This does not mean the website processes no technical information. Vercel receives ordinary network requests to host, protect, and measure aggregate use and performance of the site. That processing and its legal basis are explained in the Privacy Notice.

4. External websites

If you follow a link to Discord, open an email application, or visit another external service, you leave this website. Those services may use their own cookies or similar technologies under their own policies and controls. Project Caligo does not control cookies placed by a separate website you choose to visit.

5. If this changes

If we later add embedded media, authentication, advertising, or another feature that stores or reads information on your device, we will update this notice before or when the feature is introduced. Where applicable law requires consent, we will add a suitable consent control before activating the non-essential technology.

6. How to verify or ask

You can inspect your browser's developer tools or site-data settings to confirm what a website stores. Questions about this notice can be sent to contact@industrium.net.
